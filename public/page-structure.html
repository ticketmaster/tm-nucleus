Page Structure

The basic layout of our pages is as follows:

<body class="nc-page edp">
<div class="nc-page__wrapper">
<div class="nc-page__frame">
<div class="nc-page__content">
  <!-- page content here -->
.nc-page Sets base typography for page and background color
.nc-page__wrapper Sets background color for a particular section of the page
.nc-page__frame Sets max-width for your content area
.nc-page__content Sets default font size and other needed attributes for your content area
.[pagename] Sets block context for any page-specific styles

Note that you don't always need to use all of the classes above. Only use those classes that you need to establish page structure.

For example, if you don't need set to a background color for your section of a page that's different from the body background, then you don't need to use nc-page__wrapper. Similarly, if your content is supposed to span the full width of the browser at all times, then you don't need page__frame.
