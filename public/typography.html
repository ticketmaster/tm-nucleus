Typographic Scale

We utilize a typographic modular scale using 16px as the base size and 1.125 ratio. All of our font sizes are derived as a multiple of these two numbers. We specify our font sizes in rem units. 1rem is equivalent to 10px.

With few exceptions, font sizes on our pages are responsive at the main breakpoints.

Important Note

A developer should assume that any font size specified by a designer is responsive unless explicitly stated otherwise.

Line height

Line height is a constant 1.4 ratio of the current font size.

Fixed font sizes

There are occasions where the designer may choose to use a fixed font that is the same across all viewport sizes. In these cases, the text will render in the Small sizes only (refer to the Size chart section above).

Code Reference

nc-type-scale() Mixin mixins/_text.scss

Generates font size from our typographic scale.

Parameters Default Description Required
$set   Name of size set (refer to Size Chart)
$lh $nc-base-line-height Line height  
$responsive true Should the font be responsive   
/* basic usage */
.text-basic {
  @include nc-type-scale('kilo');
}

/* passing line-height */
.text-lh {
  @include nc-type-scale('kilo', $lh: 1);
}

/* fixed font size aka not responsive */
.text-fixed {
  @include nc-type-scale('kilo', $responsive: false);
}

nc-alt-copy() Mixin mixins/_text.scss

Sets text color to Cascade and font weight to normal.

Parameters Default Description Required
$set false Name of size set (refer to Size Chart)  
$lh $nc-base-line-height Line height  
$responsive true Should the font be responsive. Only applies if $set is specified.   
/* basic usage */
.alt-basic {
  @include nc-alt-copy();
}

/* specify a font size that's different from its context */
.alt-size {
  @include nc-alt-copy($set: 'centi');
}

/* fixed font size aka not responsive */
.alt-fixed {
  @include nc-alt-copy($set: 'centi', $responsive: false);
}

nc-all-caps() Mixin mixins/_text.scss

Set letter-spacing and transform text to uppercase.

Parameters Default Description Required
$ls $nc-caps-letter-spacing Amount of letter spacing between characters   
/* basic usage */
.all-caps {
  @include nc-all-caps();
}

/* specify custom letter-spacing */
.all-caps-custom {
  @include nc-all-caps($ls: 5px);
}

nc-truncate-text() Mixin mixins/_text.scss

Sets text to show an ellipsis when length is beyond specified width.

Important Note

Prior to 1.0.0, this was a parameter-less mixin. To remain backwards-compatible, the new $width parameter is set to false by default. Going forward, all future invocations of this mixin should pass in the width as a parameter.

Parameters Default Description Required
$width false Width of text area to show before ellipsis 
/* basic usage */
.truncated-text {
  @include nc-truncate-text(200px);
}

nc-get-font() Function mixins/_text.scss

Returns font-size given the name and size from named set.

Parameters Default Description Required
$name   Name of size (refer to Size Chart)
$size   Size from named set (sm, md, lg) 
/* basic usage */
.some-text {
  font-size: nc-get-font('mega', 'md');
}

/* output */
.some-text {
  font-size: 2rem;
}
