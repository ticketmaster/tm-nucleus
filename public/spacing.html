Scale

Whitespace is an important part of creating rhythm and harmony between elements on a page. To maintain consistency, we've developed a spacing scale that has an 8px base size and a 1.5 ratio. All of our spacing values are derived as a multiple of these two numbers.

With few exceptions, font sizes on our pages are responsive at the main breakpoints.

Code Reference

nc-spacing() Mixin mixins/_spacing.scss

Generates size value from our spacing scale.

Parameters Default Description Required
$outside false Set margins; can be either name from size chart or a SASS map  
$inside false Set padding; can be either name from size chart or a SASS map  
$responsive true Should the size be responsive   
/* basic usage */
.widget {
  @include nc-spacing($inside: compact);
}

/* using a SASS map; can either define separately or inline within @include call */
$map: (
  top: 10px,
  right: slim,
  left: 0
);

.widget-custom {
  @include nc-spacing($outside: $map);
}

/* fixed spacing aka not responsive */
.text-fixed {
  @include nc-spacing($inside: moderate, $responsive: false);
}
